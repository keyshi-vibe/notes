$a^b (\mod n)$ 
расскалыдваем b в двоичную запись и используем следующий алгоритм

Алгоритм БЫСТРОЕ_ВОЗВЕДЕНИЕ_В_СТЕПЕНЬ(основание a, степень b, модуль n) 
1. Пусть d = 1 // начальный результат
2. Определяем двоичное представление числа b: (b_k, b_k-1, ..., b_0) 
3. Для каждого бита i от k вниз до 0:
4.  d = (d * d) mod n // шаг возведения в квадрат
5.  Если текущий бит b_i == 1: d = (d * a) mod n // шаг дополнительного умножения 
6. Вернуть d
