Объектом явлется ndarray. У любого массива есть атрибут
shape - кортеж который описывает размер по каждому измерению dtype - объект который описывает тип данных
ndim - кол-во измерений массива 
tolist( ) - конвертирует array в обычный python list

создание массива 
без явного типа данных
$arr = np.array([1,2,3,4,5])$
и с явным типом данных
$arr = np.array([1,2,3,4,5], dtype = np.int32)$

Так же тип данных можно изменить с помощью astype()
$arr = np.asstype(np.int64)$

$arrange$ принимаем на вход 3 аргумента, а именно начальный элемент, конечный элемент и шаг. Метод позволяет генеративно заполнить массив.

$linspace$ схож с $arrange$ только вместо 3 аргумента шага подается аргумент сколько нужно чисел

для генерации случайных чисел используется 
$np.random.random(кол-во аргументов)$

для генерации от диапазона $[a,b]$ используется формула
$(b - a) \cdot np.random() + a$ где $b>a$

Но рандом уже устарел поэтому сейчас нужно использовать генератор

Для работы с генератором нужно создать обьект-генератор
$$rng = np.random.default_rng() $$
$$
rng = np.random.default_rng(seed=42)$$
