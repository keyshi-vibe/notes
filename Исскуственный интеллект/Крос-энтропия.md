Кросс-энтропия измеряет расхождение между предсказанным распределением вероятностей и истинным
$$L = - \sum_{j=1}^{K}y_j ln(\hat{y_j})$$

т.к предсказание содержит вектор из нулей и одной единицы, то вся формула схлопывается в $$L = - ln(\hat{y_{correct}})$$
Т.е для всех неправильных классов сумма будет равна 0 и только для истинного класса формула выдаст ответ больше 0

- **Модели плевать на всё остальное:** Чтобы минимизировать ошибку L, модели нужно сделать так, чтобы $\hat{y}_{​correct​}$  был как можно ближе к **1**.
    
- **Логарифмический штраф:**
    
    - Если модель дала правильному классу вероятность 0.99, то −ln(0.99)≈0.01 (почти нет штрафа).
        
    - Если модель дала правильному классу вероятность 0.01 (почти не узнала его), то −ln(0.01)≈4.6 (огромный штраф!).
        
    - Если вероятность правильного класса 0, ошибка уходит в +∞.
