Идея заключается в следующем: пусть обучающая выборка состояла из n элементов. Выберем из нее n примеров равновероятно, с возвращением. Получим новую выборку $X^1$, в которой некоторых элементов исходной выборки не будет, а некоторые войдут несколько раз.  С помощью некоторого алгоритма b обучим на этой выборке модель $$b_1(x) = b(x,X^1)$$
Повторим эту процедуру $k$ раз и получим $k$ моделей, обученных на $k$ выборках. Чтобы получить одно предсказание, усредним предсказание всех моделей: $$a(x) = \frac{1}{k}(b_1(x) + ... + b_k(x))$$
Процесс генерации подвыборок с помощью семплирования с возвращением называется бустрепом, а модели $b_1(x), ...., b_k(x)$ называют базовыми моделями. Модель $a(x)$ называется ансамблем этих моделей.
