Для регуляризации используется l1 или l2
l1 это суммирование всех признаков по модулю что по итогу создает ромб
l2 это суммирование всех признаков по квадрату что по итогу образует окружность
Главное замечание: $w_0$ (а именно смещение) мы не трогаем
Обычно регуляризацию использует с MSE
L1 + MSE
L2 + MSE

Регуляризация помогает модели не переобучаться:
* модель не учитывает шум 
* модель понижает веса признаков
* ненужные веса в случае l1 просто зануляются
