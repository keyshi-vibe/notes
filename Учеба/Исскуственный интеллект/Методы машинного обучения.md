Обучение делится на ![обучение с учителем](обучение с учителем.md) и ![обучение без учителя](обучение без учителя.md)
. Если мы скормим модели все данные сразу, то она быстро переобучится. Для этого данные разделяют на 3 части:
* Обучающая выборка (примерно 60-70% датасета). На ней модель модель видит и признаки, и таргеты по которым она подбирает параметры.
* Валидационная выборка (примерно 10-20%). На ней модель не учится. Нужна только для настройки параметров
* Тестовая выборка (примерно 10-20%).  Эти данные модель никогда не видела во время обучения. По ней прогоняют в самом конце, чтобы увидеть результаты модели.

### В чем суть обучения?
Например при градиентном спуске:
1) модель берет случайные веса
2) модель высчитывает предсказание
3)  модель считает ошибку
4) в этот же момент модель считает градиентный спуск
5)  обновляем веса в сторону антиградиента (т.е. пытаемся максимально снизить ошибку)
6)  повторяем с п1 по п5 пока не попадем в минимальную оишбку

### В чем разница между обучением и предсказанием
При обучении модель имеет на входе признаки и таргеты, постоянно считает ошибку и обновляет веса. Итогом будет обновление параметров и минимизирование ошибки.
При предсказании модель имеет на вхоже только признаки, она не обновляется, не считает ошибки, не обновляет параметры. Итогом будет конкретный прогноз.

Функция ошибки MSE

$MSE(w) = \frac{1}{m} \sum_{i=1}^{m}(h_w(x_i) - y_i)^2$   где 
m - кол-во признаков 
$y_i$ - i-ый таргет (а именно истинное значение для него)
$h_w(x_i)$ - предсказание для этого же объекта

### Переобучение
Переобучение - это когда модель практически не совершает ошибок. Это не значит, что модель гений, а это значит что модель просто выучила значения датасете и при новом датасете совершит огромные ошибки.
