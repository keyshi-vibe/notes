Представим, что у нас много данных и нужно провести линию, которая будет их разделять по классам. Таких линий можно провести тысячи. Для такого случая используется метод опорных векторов.
SVM ищет линию, которая находится на максимально возможном удалении от ближайших точек обоих классов

#### Геометрическая интерпретация

Существует 3 ключевых понятия:
1) Разделяющая гиперплоскость: В 2D - это линия, в 3D - плоскость, а в 7D - это гиперплоскость. В любом из случаев уравнение для решения задачи будет $w^T \cdot x + b = 0$
2) Зазор (Margin): Это "пустая полоса" вдоль разделяющей линии. Расстояние от линии до ближайщей точки называется полушириной зазора. SVM максимизирует общую ширину этой полосы ($\frac{2}{||w||}$)
3) Опорные вектора: Это те самые точки, которые лежат ровно на границе этого зазора

#### Математическая интерпретация 

Чтобы ваша модель нашла эту границу, она решает задачу оптимизации.

Жесткий зазор. При таком зазоре мы ставим условие, что ни одна точка не должна попасть внутрь зазора
Математически мы минимизуем: $$\frac{1}{2} \cdot ||w||^2$$
при условии, что для каждой точки $i$: $$y_i(w^Tx_i+b) \geq 1$$
Мягкий зазор. Мы ставим условие, что точка может падать на зазор, но мы штрафуем модель.
Математически мы минимизиурем: $$\frac{1}{2}||w||^2 + C \sum \xi_i$$
где: 
*  Параметр C - цена ошибки. Если он большой то модель рискует переобучиться, т.к. зазор станет очень узким. Если параметр маленький, то модель позволяет некоторым точкам залезть на зазор, чтобы сама граница была ровной и стабильной

### 1. Выделение опорных векторов
Опорными векторами называются точки обучающей выборки, которые лежат на границах разделяющей полосы. Они "подпирают" зазор и определяют положение разделяющей гиперплоскости.
* Положительный класс ($+$): $\vec{x}_{pos} = \begin{pmatrix} -2 \\ 1 \end{pmatrix}$
* Отрицательный класс ($-$): $\vec{x}_{neg} = \begin{pmatrix} 1 \\ -2 \end{pmatrix}$

---

### 2. Геометрическое построение
1. **Вспомогательный вектор (нормаль) $\vec{w}$**: Направлен от отрицательного класса к положительному. Соединяем точки $(1, -2)$ и $(-2, 1)$. Направление вектора: $(-3, 3)$. Это означает, что разделяющая прямая будет перпендикулярна вектору с наклоном $-1$, то есть сама прямая пойдет под углом $45^\circ$.
2. **Решающая граница (пунктир)**: Проходит ровно посередине между опорными векторами через точку $M$:
   $$M = \frac{\vec{x}_{pos} + \vec{x}_{neg}}{2} = \left( \frac{-2+1}{2}, \frac{1-2}{2} \right) = (-0.5, -0.5)$$
3. **Границы отступа (сплошные линии)**: Проходят параллельно решающей границе непосредственно через точки $(-2, 1)$ и $(1, -2)$.

---

### 3. Математический расчет параметров $w$ и $b$

Гипотеза классификатора: $h(x) = w_1x_1 + w_2x_2 + b$.
Для опорных векторов выполняются условия:
1. $w_1(-2) + w_2(1) + b = 1$
2. $w_1(1) + w_2(-2) + b = -1$

**Находим веса $w$:**
Вычтем второе уравнение из первого:
$$(-2w_1 - 1w_1) + (1w_2 - (-2w_2)) = 1 - (-1)$$
$$-3w_1 + 3w_2 = 2 \implies 3(w_2 - w_1) = 2 \implies w_2 - w_1 = \frac{2}{3}$$

Из симметрии задачи и направления вектора $(-3, 3)$ видно, что $w_1 = -w_2$. Подставляем:
$$w_2 - (-w_2) = \frac{2}{3} \implies 2w_2 = \frac{2}{3} \implies \mathbf{w_2 = \frac{1}{3}, \ w_1 = -\frac{1}{3}}$$

**Находим смещение $b$:**
Подставим $w_1$ и $w_2$ в первое уравнение:
$$-2\left(-\frac{1}{3}\right) + 1\left(\frac{1}{3}\right) + b = 1$$
$$\frac{2}{3} + \frac{1}{3} + b = 1 \implies 1 + b = 1 \implies \mathbf{b = 0}$$

---

### 4. Итоговая гипотеза классификатора
Классификатор принимает решение на основе знака функции:
$$h(x) = \text{sign}\left( -\frac{1}{3}x_1 + \frac{1}{3}x_2 \right)$$

### 5. Ширина разделяющей полосы (зазор)
Ширина зазора $M$ вычисляется по формуле $M = \frac{2}{\|\vec{w}\|}$:
$$\|\vec{w}\| = \sqrt{\left(-\frac{1}{3}\right)^2 + \left(\frac{1}{3}\right)^2} = \sqrt{\frac{1}{9} + \frac{1}{9}} = \frac{\sqrt{2}}{3}$$
$$M = \frac{2}{\sqrt{2}/3} = \frac{6}{\sqrt{2}} = 3\sqrt{2} \approx 4.24 \text{ единицы}$$
