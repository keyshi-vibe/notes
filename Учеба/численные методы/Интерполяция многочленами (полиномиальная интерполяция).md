Задача сходится к нахождению многочлена n-ой степени P$_n$(x), удовлетворяющего условиям
P$_n$(x$_i$) = y$_i$  $i$ = 0, ... , n
Такой многочлен единственен, если все точки x$_i$ различны
Для построения такого многочлена может быть использована ![интерполяционная формула Лагранжа](интерполяционная%20формула%20Лагранжа.md)

Более эффективным построение будет через ![интерполяционная формула Ньютона](интерполяционная%20формула%20Ньютона.md) 

### Погрешность полиномиальной интерполяции
Погрешность $R_n(x) = f(x) - P_n(x)$ может быть оценена по формуле:
$$|R_n(x)| \le \frac{M_{n+1}}{(n+1)!} |x - x_0| \cdot |x - x_1| \dots |x - x_n|, \quad (8)$$
здесь $M_{n+1} = \max\limits_{\xi \in [a,b]} |f^{(n+1)}(\xi)|$, а $[a,b]$ — минимальный отрезок, содержащий точку $x$ и все узлы интерполяции.
